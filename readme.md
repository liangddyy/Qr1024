# 全能扫码

很久之前课程设计做的，一个简单的二维码的应用。

包括

1. 二维码生成、识别
2. 读取手机联系人生成二维码、扫码自动添加到联系人
3. 读取手机WiFi信息（需root）、扫码自动连接WiFi
4. 历史记录保存到Sqlite。
